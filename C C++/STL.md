# STL

## 组成

- 容器（containers）
- 算法（algorithms)
- 迭代器（iterators）
- 仿函数（function）
- 配接器（adapter）
- 空间配置器（allocator）

## 容器（containers）

1. 序列式容器（sequence containers)：元素都是可序（ordered）,但未必是有序（sorted），如：array，vector
2. 关联式容器（associattive containers)：如：list，map

array是固定大小的容器，它们保存了一个以严格的线性顺序排列的特定数量的元素。
